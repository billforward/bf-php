To get the workspace setup, there are only three steps:

1. Install java, curl and php, and add them to your PATH.
2. Install gradle
3. Install Composer


=Step 1. Installing Java and PHP.=
These are well-documented; follow normal installation process.

=Step 2. Installing gradle.=
(Requires Java on PATH).
We provide a Gradle self-installer for Windows and UNIX.
Invoke with:

gradlew

=Step 3. Installing Composer.=
(Requires PHP on PATH).
We provide a Composer self-installer for Windows and UNIX.
Invoke with:

gradlew -q setup
